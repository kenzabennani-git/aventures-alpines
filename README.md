# Aventures Alpines

Plateforme web dediee aux sports de montagne - BTS SIO SLAM.

## Description

Site web pour explorer et partager des experiences autour des sports de montagne : randonnee, escalade et ski.

## Fonctionnalites

- Page d'accueil avec activites populaires
- Pages dediees : Randonnee, Escalade, Ski
- Blog avec articles
- Connexion / inscription utilisateur
- Espace administrateur
- Formulaire de contact
- API REST (Node.js / Express)
- Base de donnees MySQL

## Technologies

- React (frontend)
- Node.js / Express (backend)
- MySQL (base de donnees)
- Axios (requetes HTTP)
- JavaScript
